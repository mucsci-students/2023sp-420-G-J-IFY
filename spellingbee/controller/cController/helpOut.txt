------------------------------------------------------------
|                       How to Play                        |
|----------------------------------------------------------|
| Simply type a word after   |  To enter a command, simply |
| the '> ' prompt and press  |  type the command, or hit   |
| enter to submit a guess    |  TAB to cycle through them  |
|----------------------------------------------------------|
|                    Available Commands                    |
|----------------------------------------------------------|
|     Command     |             Description                |
|----------------------------------------------------------|
| shuffleLetters  | Shuffle the order of the active puzzle |
|                 | for a fresh view                       |
|----------------------------------------------------------|
| showHints       | Displays some helpful hints to solve   |
|                 | the puzzle                             |
|----------------------------------------------------------|
| showFoundWords  | Displays the list of all               |
|                 | discovered words                       |
|----------------------------------------------------------|
| showLeaderboard | Display the leaderboard with the       |
|                 | top 10 scores for this puzzle          |
|----------------------------------------------------------|
| saveGame        | Create a new save for the current game |
|----------------------------------------------------------|
| saveBlankGame   | Create a new save for a blank version  |
|                 | of the game (does not include any      |
|                 | progress from current session)         |
|----------------------------------------------------------|
| endGame         | Exits the current game to main menu.   |
|                 | Checks if you make the leaderboard     |
------------------------------------------------------------
